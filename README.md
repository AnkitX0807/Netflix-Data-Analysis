# 🎬 Netflix Data Analysis (EDA Project)

## 📌 Project Overview

This project performs an Exploratory Data Analysis (EDA) on the Netflix dataset to uncover patterns in content distribution, genres, ratings, and global expansion trends.

The analysis focuses on transforming raw data into meaningful insights using Python and visualization techniques.



## 📂 Dataset

* Source: Kaggle
* Dataset: Netflix Movies and TV Shows
* Contains information such as:

  * Title, Director, Cast
  * Country, Release Year
  * Rating, Genre
  * Date Added



## 🛠️ Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab


## 🔍 Key Analysis Performed

### 🧹 Data Cleaning

* Handled missing values using appropriate strategies
* Converted date columns into datetime format
* Extracted Year, Month, and Day features



### 📊 Exploratory Data Analysis

* Distribution of Movies vs TV Shows
* Most common genres
* Country-wise content production
* Most frequent actors and directors
* Rating distribution



### 📈 Time-Based Analysis

* Content addition trends over the years
* Monthly content addition patterns
* Peak years of Netflix growth



### 🌍 Global Analysis

* Country-wise contribution over time
* Analysis of Netflix's global expansion strategy



## 💡 Unique Approach

* Used **split + explode technique** to handle multi-value columns (genres, cast, country, director)
* Performed **multi-variable analysis** (country vs year, type vs rating)
* Focused on **data cleaning before visualization** for accurate insights



## 📊 Key Insights

* Movies dominate Netflix content compared to TV Shows
* International content is highly prevalent across genres
* The United States contributes the most content, followed by India and the UK
* Content additions increased rapidly after 2015, peaking around 2019
* Most content is rated TV-MA, indicating a focus on mature audiences
* Netflix primarily features modern content (post-2000)
* Frequent appearances of certain actors and directors suggest strong collaborations



## 🏁 Conclusion

Netflix has significantly expanded its content library over time, especially after 2015. The platform has evolved into a global content provider, focusing on diverse, modern, and mature content to cater to a wide international audience.
